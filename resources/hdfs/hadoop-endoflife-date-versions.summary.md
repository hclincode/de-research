## Summary

The endoflife.date community-maintained tracker for Apache Hadoop versions documents which release lines are actively supported vs. end-of-life. The tracker sources data from the official Apache Hadoop Active Release Lines wiki page. As of May 2026, Apache Hadoop has 4 supported release lines with 3.5.0 as the latest stable release (April 2026). The 3.2 branch reached EOL on December 21, 2023. Apache Hadoop does not publish a fixed release and support policy, which creates uncertainty for organizations planning long-term deployments.

## Key Points

- Apache Hadoop 3.5.0 (April 2026) is the current latest stable release.
- 4 active release lines as of May 2026.
- Hadoop 3.2 reached EOL: December 21, 2023.
- 14 Apache Hadoop release lines have reached End of Life.
- Apache Hadoop does NOT have a formal, time-bound support policy — EOL dates are set ad-hoc by the community.
- Authoritative source for EOL dates: Apache Hadoop Active Release Lines wiki (cwiki.apache.org/confluence/display/HADOOP/Hadoop+Active+Release+Lines).
- Implication: organizations cannot plan around guaranteed Hadoop LTS windows the way they can with products like Red Hat or Ubuntu.
- 3.6.0-SNAPSHOT is in active development (last publish May 15, 2026).
