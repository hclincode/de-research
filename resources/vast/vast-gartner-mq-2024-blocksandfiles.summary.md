## Summary

Blocks & Files coverage of the October 2024 Gartner Magic Quadrant for File and Object Storage Platforms, reporting that VAST Data was named a Leader (having moved from Challenger in 2023). Weka is not mentioned as being in the 2024 MQ. Cloudian, DDN, NetApp, and Quantum were dropped from the evaluation for "not meeting the 2024 inclusion criteria of a single platform for file and object workloads."

## Key Points

- **VAST Data**: Named **Leader** in 2024 Gartner MQ for File and Object Storage Platforms (moved up from Challenger in 2023)
- **Other Leaders**: Dell (top position), Pure Storage
- **Also included**: Scality (Visionary), Hitachi Vantara (Niche), DataCore
- **Weka**: Not mentioned in this article — not confirmed as included or excluded from the 2024 MQ
- **Dropped vendors**: Cloudian, DDN, NetApp, Quantum — for not meeting the unified file+object platform criterion
- **Criterion shift**: Gartner 2024 MQ emphasized unified file-and-object platforms, disadvantaging pure object or pure file vendors
- **VAST Gartner Peer Insights**: 4.9/5, 98% willingness to recommend (60+ reviews as of October 2024)
- **Practical implication**: VAST's Leader status signals enterprise analyst recognition; Weka's absence from the MQ is a market-position gap for procurement teams using Gartner evaluation criteria
